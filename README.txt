# Complejidad-Computacional
Repositorio destinado a las praticas de la asignatura CC.

En este repositorio se encuentran las practicas 2 y 3 de la asignatura Complejidad Computacional. Ambas son basadas en el uso de grafos
para representar el problema.
Estan realizadas en C++ y ejecutadas mediante un projecto en Code::Blocks.
